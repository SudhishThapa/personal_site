
# Sudhish Hub (Flask)

Simple Flask site with sections: Basketball, Music, Japanese, Finance, General Learning.

## Run locally
1) python -m venv .venv
2) Activate: .venv\\Scripts\\Activate.ps1 (Windows) or source .venv/bin/activate (Mac)
3) pip install -r requirements.txt
4) Set env vars: ADMIN_PASSWORD and SECRET_KEY
5) python app.py
